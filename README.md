## ğŸš€ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS)
- [yt-dlp](https://github.com/yt-dlp/yt-dlp#installation) installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/HarryMofoka/Flux-Drop.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
